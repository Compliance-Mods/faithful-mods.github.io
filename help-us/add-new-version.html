Add a new Minecraft version support:

Prerequisites:

You need to have an GitHub account first.
It is recommended to download GitHub Desktop for easier file and branch management.

  1. Firstly, check if the branch already exist, if not create a new one :

    GitHub:

    Type the Minecraft version you want add support for, then click on Create a branch.

    GitHub Desktop:

    Click on the current branch button, then New branch, then type the Minecraft version you want add support for, if main is not selected, click on it, then create a new branch.

  2. Add those files to your new branch:

    • pack.png

    • pack.mcmeta with this inside:

      
{
	"pack":{
		"pack_format": 3,
		"description": "Faithful Mods"
	}
}

      The "pack_format" can be found here (make a pull request if outdated).

    • An assets folder with mods textures insides, following this path: assets//textures/...

  3. If the default file from the main branch (file nammed: initialcommit) is present, remove it. Then if you are not a member of our organization, make a pull request (do not merge the branch: never) & post a message in our Discord
