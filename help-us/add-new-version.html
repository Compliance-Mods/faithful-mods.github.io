Add a new Minecraft version support:

Prerequisites:

You need to have an GitHub account first.
It is recommended to download GitHub Desktop for easier file and branch management.

  1. Firstly, check if the branch already exist, if not create a new one :

    GitHub:

    Type the Minecraft version you want add support for, then click on Create a branch.

    GitHub Desktop:

    Click on the current branch button, then New branch, then type the Minecraft version you want add support for, if main is not selected, click on it, then create a new branch.
